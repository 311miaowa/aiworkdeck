# 阿里云 ECS 安全组配置说明

## 需要开放的端口

### 必须开放的端口（公网访问）

| 端口 | 协议 | 说明 | 来源 |
|------|------|------|------|
| **22** | TCP | SSH 远程管理 | 你的 IP 地址（建议限制）或 0.0.0.0/0（不推荐，但方便） |
| **80** | TCP | HTTP（用于申请 SSL 证书） | 0.0.0.0/0 |
| **443** | TCP | HTTPS（API 访问） | 0.0.0.0/0 |

### 不需要开放的端口（仅内网访问）

| 端口 | 协议 | 说明 | 原因 |
|------|------|------|------|
| **9696** | TCP | 后端 Spring Boot 服务 | **只允许内网访问**，通过 Nginx 反向代理暴露，不直接暴露到公网 |
| **3306** | TCP | MySQL 数据库 | 只允许内网访问，不暴露到公网 |

## 安全组配置步骤

### 在阿里云控制台配置

1. **登录阿里云控制台**
   - 进入 ECS 实例列表
   - 找到你的 ECS 实例

2. **配置安全组**
   - 点击实例 → 安全组 → 配置规则
   - 或直接进入：网络与安全 → 安全组 → 找到你的安全组 → 配置规则

3. **添加入站规则**

   **规则 1：SSH (22)**
   - 端口范围：`22/22`
   - 协议类型：`TCP`
   - 授权对象：`你的IP地址/32`（推荐，更安全）
     - 或者 `0.0.0.0/0`（允许所有 IP，方便但不安全）
   - 描述：`SSH 远程管理`

   **规则 2：HTTP (80)**
   - 端口范围：`80/80`
   - 协议类型：`TCP`
   - 授权对象：`0.0.0.0/0`
   - 描述：`HTTP，用于申请 SSL 证书`

   **规则 3：HTTPS (443)**
   - 端口范围：`443/443`
   - 协议类型：`TCP`
   - 授权对象：`0.0.0.0/0`
   - 描述：`HTTPS，API 访问`

4. **确认出站规则**
   - 默认允许所有出站流量即可（用于下载依赖、调用外部 API 等）

## 安全建议

### 1. SSH 端口限制（强烈推荐）

**不要**将 SSH (22) 端口开放给 `0.0.0.0/0`，应该：
- 只开放给你的固定 IP 地址
- 或者使用 VPN/跳板机访问

**如何获取你的 IP：**
- 访问 https://www.ipip.net/ip.html
- 复制显示的 IP 地址
- 在安全组中设置为：`你的IP/32`

### 2. 后端端口不暴露

**9696 端口绝对不要开放到公网**，原因：
- 后端服务没有认证机制（目前）
- 直接暴露容易被攻击
- 通过 Nginx 反向代理可以提供：
  - SSL 加密
  - 访问控制
  - 日志记录
  - 限流保护

### 3. 数据库端口不暴露

**3306 端口不要开放到公网**，原因：
- MySQL 只允许内网访问
- 如果需要远程管理，使用 SSH 隧道

## 验证配置

### 1. 检查端口是否开放

```bash
# 在本地测试（应该能访问）
curl http://你的域名/api/projects

# 测试 9696 端口（应该无法访问，说明配置正确）
telnet 你的ECS_IP 9696  # 应该连接失败
```

### 2. 检查后端是否正常运行

在 ECS 服务器上：

```bash
# 应该能访问（内网）
curl http://localhost:9696/api/projects

# 检查端口监听
netstat -tlnp | grep 9696
```

## 端口说明总结

```
公网访问：
├── 22 (SSH)      ← 只开放给你的 IP
├── 80 (HTTP)     ← 开放给所有人（申请 SSL 用）
└── 443 (HTTPS)   ← 开放给所有人（API 访问）

内网访问（不开放到公网）：
├── 9696 (后端)   ← 只允许 127.0.0.1 访问，通过 Nginx 代理
└── 3306 (MySQL)  ← 只允许 127.0.0.1 访问
```

## 架构说明

```
用户浏览器
    │
    │ HTTPS (443)
    ▼
┌─────────────────┐
│  Nginx (宝塔)   │  ← 监听 80/443
│  反向代理        │
└────────┬────────┘
         │ HTTP (内网)
         │ localhost:9696
         ▼
┌─────────────────┐
│  Spring Boot    │  ← 监听 9696（仅内网）
│  后端服务        │
└────────┬────────┘
         │ JDBC (内网)
         │ localhost:3306
         ▼
┌─────────────────┐
│  MySQL          │  ← 监听 3306（仅内网）
└─────────────────┘
```
